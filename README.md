# Represente-me! | Hub de Integrações

Este é o projeto **Represente-me!**, um hub de integrações com foco em alta performance e design minimalista premium.

## 🚀 Como Iniciar

Siga os passos abaixo no seu terminal para rodar o projeto localmente:

1. **Instalar dependências** (apenas na primeira vez):
   ```bash
   npm install
   ```

2. **Rodar em desenvolvimento**:
   ```bash
   npm run dev
   ```

3. **Gerar build de produção**:
   ```bash
   npm run build
   ```

## 📂 Estrutura do Projeto (Sênior)

- `app.js` - Core da aplicação (Manipulação DOM, Mapas, Fluxos).
- `login.js` - Lógica de autenticação e validação.
- `index.html` / `landing.css` - Landing Page institucional.
- `login.html` / `login.css` - Interface de acesso.
- `painel.html` / `style.css` - Dashboard principal e sistema de design.
- `dist/` - Arquivos otimizados para produção (gerados via build).
- `node_modules/` - Dependências do projeto.

## 🎨 Sistema de Design

O projeto utiliza uma estética **Minimalista Premium** inspirada por plataformas como Linear e Figma:
- **Cores**: Fundo `#0A0A0A`, Acento `#FF4D00`.
- **Tipografia**: `Geist Sans` para interface, `Geist Mono` para dados técnicos.
- **Espaçamento**: Sistema de grid baseado em 8px.
